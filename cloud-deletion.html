Удаление облачных данных

Как удалить данные из облака

1 Откройте приложение Lyrica на вашем устройстве
2 Перейдите в раздел "Профиль" (иконка профиля в правом верхнем углу)
3 Прокрутите вниз до раздела "Облачные данные"
4 Нажмите на "Удалить облачные данные"
5 Подтвердите удаление данных из облака
6 Альтернативно, отправьте запрос на удаление облачных данных по электронной почте:
iva28app@gmail.com

Какие данные будут удалены из облака

  • Все тексты песен и заметки из облака
  • Быстрые заметки из облака
  • Теги из облака
  • Аудио файлы (голосовые заметки) из облака
  • Данные синхронизации

Что сохранится

✓ Ваш аккаунт останется активным
✓ Email и профиль пользователя сохранятся
✓ Локальные данные на устройстве не будут затронуты
✓ Вы сможете продолжить использование приложения

Сроки обработки запроса

Запрос на удаление облачных данных будет обработан в течение 7 дней с момента получения. После удаления восстановление данных из облака невозможно.

⚠️ Важно: После удаления облачных данных вы не сможете синхронизировать данные между устройствами. Локальные данные на вашем устройстве останутся нетронутыми.

Cloud Data Deletion

How to Delete Cloud Data

1 Open the Lyrica app on your device
2 Go to the "Profile" section (profile icon in the top right corner)
3 Scroll down to the "Cloud Data" section
4 Tap "Delete Cloud Data"
5 Confirm deletion of cloud data
6 Alternatively, send a cloud data deletion request by email:
iva28app@gmail.com

What Cloud Data Will Be Deleted

  • All song lyrics and notes from cloud
  • Quick notes from cloud
  • Tags from cloud
  • Audio files (voice notes) from cloud
  • Synchronization data

What Will Be Preserved

✓ Your account will remain active
✓ Email and user profile will be preserved
✓ Local data on your device will not be affected
✓ You can continue using the app

Processing Time

Cloud data deletion requests will be processed within 7 days of receipt. After deletion, cloud data recovery is not possible.

⚠️ Important: After deleting cloud data, you will not be able to synchronize data between devices. Local data on your device will remain untouched.
